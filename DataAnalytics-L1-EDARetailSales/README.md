# 📊 Retail Sales Exploratory Data Analysis (EDA)

## 📌 Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on a Retail Sales dataset to identify sales trends, customer behavior, product performance, and business insights using Python.

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 📂 Project Files

- Thamanna_Task2_RetailEDA.ipynb
- retail_sales_dataset.csv
- README.md

## 📊 Project Workflow

- Data Loading
- Data Exploration
- Data Cleaning
- Descriptive Statistics
- Time Series Analysis
- Product Analysis
- Customer Analysis
- Data Visualization
- Business Insights
- Conclusion

## 📈 Visualizations

- Sales Trend
- Product-wise Sales
- Category-wise Sales
- Customer Analysis
- Correlation Heatmap

## ✅ Conclusion

This project analyzes retail sales data to identify important business trends and provides insights that can help improve sales performance and decision-making.
